===============
Non-member swap
===============

.. code:: cpp

    template <typename T, typename Allocator>
    void swap( concurrent_bounded_queue<T, Allocator>& lhs,
               concurrent_bounded_queue<T, Allocator>& rhs );

Equivalent to ``lhs.swap(rhs)``.
