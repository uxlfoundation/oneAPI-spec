============
continue_msg
============
**[flow_graph.continue_msg]**

An empty class that represents a continue message. An object of this class is used to indicate that
the sender has completed.

.. code:: cpp

    // Defined in header <oneapi/tbb/flow_graph.h>

    namespace oneapi {
    namespace tbb {
    namespace flow {

        class continue_msg {};

    } // namespace flow
    } // namespace tbb
    } // namespace oneapi
